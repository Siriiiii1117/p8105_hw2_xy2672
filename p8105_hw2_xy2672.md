## Problem 1

Read the `pols_month` datafile and clean

``` r
pols_month_df = 
  read_csv("data/pols-month.csv") |> 
  janitor::clean_names() |> 
  separate(mon, into = c("year", "month", "day"), sep = "-") |> 
  mutate(
    month = month.name[as.numeric(month)],
    president = 
      case_match(
        prez_gop,
        1 ~ "gop",
        0 ~ "dem"
      ),
    president = as.factor(president)
  ) |> 
  select(-prez_gop, -prez_dem, -day)
```
